Jammming: Spotify Playlist Creator
===================


Project Description
-------------------

Jammming is a Spotify playlist creator that uses the Spotify OAuth flow to connect to a user’s account. Users can search for songs, add them to a playlist, and save it directly to the official Spotify app or website.


Run this app live
-------------------


Run Locally
-------------------
Local setup not supported due to Spotify API restrictions.

Features
-------------------
- Uses OAuth and PKCE to retrieve access token
- Built with React hooks for state management (no Redux or other libraries)
- A user can:
    - Search for songs
    - View results including song title, artist, and album name
    - Add them to a playlist
    - Name/rename the playlist
    - Remove songs from the playlist
    - Save the playlist directly to the official Spotify app or website

 
Updates
-------------------
- Updated authentication from the Implicit Grant Flow to PKCE, ensuring compliance with Spotify API’s latest security requirements.

Tech Stack
-------------------
- HTML, CSS, JavaScript
- React (with hooks for state management)
- Spotify Web API
